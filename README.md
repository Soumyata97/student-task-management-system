# Student Task Management System

A mini project developed using the Laravel MVC Framework with PostgreSQL database integration.  
This system allows students to manage their daily tasks efficiently with authentication, task prioritization, analytics dashboard, and clean user interface.

---

# Features

- User Authentication (Login/Register)
- Create, Edit, Delete Tasks
- Task Status Management
- Task Priority Levels
- Due Date Tracking
- Dashboard Analytics
- Pagination System
- Search Functionality
- PostgreSQL Database Integration
- Clean Responsive UI
- MVC Architecture Implementation

---

- Laravel 12

- PostgreSQL

- HTML5
- CSS3
- Bootstrap
- JavaScript

# Database Used

PostgreSQL Database Management System

# Project Structure

This project follows the MVC (Model View Controller) architecture:

- Models → Handle database logic
- Views → User Interface
- Controllers → Business logic and request handling
